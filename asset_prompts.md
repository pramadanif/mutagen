## Gemini Tips

1. **Always upload a reference image** — attach `labtube.png` and say *"Create a new sprite in this exact same pixel art style:"*
2. **If Gemini adds text/frames:** Append *"Do not add any text, labels, titles, descriptions, card borders, or UI elements. Output only the sprite object on a plain white background."*
3. **If style is too smooth:** Append *"Make the pixels more visible and blocky. Use hard edges, no anti-aliasing, no gradients."*
4. **If background is grey:** Append *"Use pure white (#FFFFFF) background."*
5. **Generate at 1024x1024** for best quality.
